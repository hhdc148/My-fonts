## 🚀 使用说明

### Minecraft Pocket Edition APK
- **版本**: 0.1.1 (经典怀旧版本)
- **平台**: Android
- **安装**: 下载 APK 后，在 Android 设备上允许"未知来源"安装即可使用

### Minecraft Seven 字体
- **格式**: OpenType (.otf)
- **用途**: 可用于设计、海报、Minecraft 风格相关项目
- **安装**: 
  - Windows: 右键字体文件 → 安装
  - macOS: 双击字体文件 → 安装字体
  - Android/iOS: 使用第三方字体管理器
